Project Highlights

  • Developed autonomous robot system capable of mapping unknown environments using sensor fusion of camera, sonar, and IR sensors
  • Implemented PID-controlled wall-following algorithm achieving consistent 10cm path accuracy
  • Created wireless data transmission system using ESP32 and MQTT for real-time map generation
  • Built object detection system using OpenMV camera and AprilTags for precise positioning
  • Designed robust navigation system combining odometry and sensor feedback for accurate return-to-start
  • Achieved successful mapping of both simple and complex environmental features
