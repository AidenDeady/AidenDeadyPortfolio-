Project Highlights

  • Developed comprehensive testing environment in Unity for evaluating robot control interfaces
  • Implemented and compared PS4 DualShock and Steam controller mappings for complex robotic manipulation tasks
  • Designed series of standardized tasks including navigation, object manipulation, and precision control
  • Conducted human subject study with statistical analysis of performance metrics
  • Collected and analyzed quantitative data on completion times, error rates, and user satisfaction
  • Generated insights on optimal interface design for robotic control systems
